How does lossy compression work?

The aim of lossy file compression is to reduce to size of the
file being compressed. It does this by removing things from the
file that is deemed useless and will not affect the file that much
Lossy compression can reduce to the file size by up to 80% of the
original size of the file, whith minimal noticeable effects.[5]

Why Use Lossy File Formats?

Lossy compression enables even better file compression than lossless
files by removing some of the information of the audio and simplifying
the data. Of course, when doing this there will be a reduction in the
quality of the audio, but by using techniques, only the audio that is not
as important as the rest will be removed.[3] Lossy files compression is
good to use when you need a file size to be very low and if it does not
matter if anything if removed from the audio.
